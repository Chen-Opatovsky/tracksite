# Squirrel Tracker

Squirrel Tracker is an application to track squirrels' location, activites and behavior in Central Park, NY. 

## Built with
- Ubuntu 18.04 LTS
- Python 3.7
- Django 2.2.7
- html5
- Bootstrap 4
- sqlite3 
- pytz 2019.3
- sqlparse 0.3.0
